# Pronunciation Taichel - 44 English Sounds

Website luyện phát âm 44 âm tiếng Anh, có URL riêng cho từng âm.

## Chạy local

```bash
npm install
npm run dev
```

Mở:

```text
http://localhost:3000
http://localhost:3000/sound/long-i
http://localhost:3000/sound/p
http://localhost:3000/sound/theta
```

## Deploy lên Vercel

1. Tạo tài khoản Vercel.
2. Upload toàn bộ thư mục này lên GitHub hoặc dùng Vercel CLI.
3. Vercel tự nhận NextJS.
4. Gắn domain/subdomain, ví dụ:
   - pronunciation.taichel.vn
   - english.taichel.vn

## URL riêng từng âm

Dạng:

```text
https://your-domain.com/sound/long-i
https://your-domain.com/sound/short-i
https://your-domain.com/sound/long-u
https://your-domain.com/sound/p
https://your-domain.com/sound/theta
```

## Chế độ sửa

Trong website bấm “Bật/Tắt chế độ sửa” để sửa:
- Thông tin âm
- Hướng dẫn
- Từ đơn
- Cụm từ
- Câu
- Minimal pairs

Dữ liệu sửa lưu trong trình duyệt bằng localStorage. Có nút “Xuất dữ liệu JSON” để sao lưu.
